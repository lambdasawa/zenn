普段自分が愛用している Tridactyl という Firefox の拡張機能について紹介します。

Tridactyl は以下のような機能をもっています。

- Vim ライクなキーバインドで Firefox を操作できる
- Vim ライクな設定ファイルでキーバインドを管理できる
- Vim ライクな設定ファイルでコマンドを定義できる
